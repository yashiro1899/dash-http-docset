The HTTP 501 Not Implemented server error response code indicates that the request method is not supported by the server and cannot be handled. The only methods that servers are required to support (and therefore that must not return this code) are {{HTTPMethod("GET")}} and {{HTTPMethod("HEAD")}}.

Note that a 501 error is not something you can fix, but requires a fix by the web server you are trying to access.

A 501 response is cacheable by default.
