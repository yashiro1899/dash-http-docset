The Content-Security-Policy header fields specify a page's sources by resource type. 

[Intro]

The HTTP response headers

This section lists the HTTP response headers that servers send back to specify allowed origins for page contents asdefined by the Content Security Policy specification.

Content-Security-Policy

Specifies a page's content origins by resource type for enforcement by the user agent. 

Content-Security-Policy-Report-Only

Specifies a page's content origins by resource type for monitoring by the server.
